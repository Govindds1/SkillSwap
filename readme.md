# SkillSwap

SkillSwap is a modern skill-sharing and networking platform where users can connect, chat, showcase skills, and collaborate with others based on shared interests and expertise.

Built using Next.js, TypeScript, Supabase, Clerk Authentication, and Tailwind CSS.

---

# Features

- User Authentication with Clerk
- User Profiles
- Real-time Messaging
- Skill Discovery
- Explore Users
- Responsive UI
- File Upload Support
- Modern Dashboard Interface
- Secure Backend Integration with Supabase

---

# Tech Stack

## Frontend

- Next.js 16
- React
- TypeScript
- Tailwind CSS

## Backend & Services

- Supabase
- Clerk Authentication
- UploadThing

# Installation & Setup

## 1. Clone Repository

```bash
git clone <repository-url>

```

---

## 2. Install Dependencies

```bash

```

Create a file named:
